# OS Course Design

## 1. Download

```bash
git clone https://github.com/zsz12251665/os-course-design.git
```

## 2. Compile

```bash
cd os-course-design
cmake .
make
```

## 3. Run

```bash
./my-os
```

## 4. Commands

- `createFile` or `new`
- `deleteFile` or `rm`
- `createDir` or `mkdir`
- `deleteDir` or `rmdir`
- `changeDir` or `cd`
- `dir` or `ls`
- `cp` or `copy`
- `sum` or `df`
- `cat` or `print`
- `exit` or `quit`
